Github 免費架站術！輕鬆打造個人品牌

老師真的佛心來著哈哈

若是完全不懂程式語言的人 非常推薦

若是稍微有學過html,css 但也沒有厲害到可以自己架設網站的捧友

這堂課絕對也是你的首選！

老師教你怎麼使用hugo(選網站主題)+github(自己的免費網域)+md語法(像寫日記的寫文章)

絕對物超所值啦！

時間也不長 剛好疫情在家沒事可以學一下~

課程購買最划算的方式↓↓↓

step1. 先註冊申請shopback獲取11.5%現金回饋

點選我的好友推薦註冊，購買課程後可以得到100元現金回饋！

shopback現金回饋是儲存在帳號的帳戶裡，需要再提領哦

因為提領要手續費，通常都會累積一定金額再拿出來

而且Foodpanda外送回饋也超高！

  • 趕快點這裡來註冊！

step2. 點選好友推薦來註冊hahow帳號

當朋友成功透過連結註冊，就會獲得 30,000 HP(hahow幣)，並可用來折抵課程高達 NTD$300！

  • 趕快點這裡來註冊！

step3. 註冊完直接購買課程，就大功告成啦！

可以開始馬上學習建立個人網站囉！
